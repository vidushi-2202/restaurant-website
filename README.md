# L'Étoile Restaurant Website

A premium luxury restaurant website built using React, Vite, Tailwind CSS, and Framer Motion.

## Features

- Responsive Design
- Luxury UI
- Smooth Animations
- Reservation Form
- Gallery
- Testimonials
- FAQ
- Contact Section

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- React Icons
